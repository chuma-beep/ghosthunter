GHOST HUNTER

A retro first-person shooter built in Go with Ebitengine

Click game then press SPACE to start

Features

5 Levels
Haunted halls, wizard dens, labyrinths, arenas and a boss chamber
3 Weapons
Pistol, Shotgun, and Machinegun with Doom-style state machine
5 Enemy Types
Ghosts, Wizards, Demons, Wraiths, and Reapers — each with unique AI
Custom Raycaster
Software-rendered 3D engine written from scratch in pure Go

Controls

Arrow Up / DownMove
Arrow Left / RightTurn
SpaceShoot
1 / 2 / 3Weapons
ESCPause
QQuit

How to Play

Runs in your browser via WebAssembly. Click the game window to focus it, then press SPACE to start.

Works best in Chrome, Firefox, or Edge.
