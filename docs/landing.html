GHOST HUNTER

A retro first-person shooter built in Go with Ebitengine

Press SPACE to start

Features

5 Levels
Progress through haunted halls, wizard dens, and more
3 Weapons
Pistol, Shotgun, and Machinegun with Doom-style animations
5 Enemy Types
Ghosts, Wizards, Demons, Wraiths, and Reapers
Custom Raycaster
Software-rendered 3D engine in pure Go

Controls

Arrow KeysMove
SpaceShoot

How to Play in Browser

This game runs in your browser using WebAssembly. Click on the game above to start. Use arrow keys to move, space to shoot.

Works best in Chrome, Firefox, or Edge.
